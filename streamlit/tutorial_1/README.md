## Requirements
- Python 3.10

## Setup instructions
1. Install the required packages
    ```bash
    pip install -r requirements.txt
    ```
2. Navigate to the directory that contains app.py
    ```bash
    cd streamlit/tutorial_1
    ```
3. Run the app
    ```bash
    streamlit run app.py
    ```
